Generates a repository that shows an image in the GitHub contributions summary.

Create your own:

You can import your own images! They just need to:

  • be 7 pixel high, and less than 52 pixel wide
  • use exactly 5 different colors

HowTo

  1. Download the sourcecode
  2. Create a new repository (for example "paper") on GitHub and initialize this repository with a README. You can use an existing repository, but you really sould not.
  3. Modify the variables in the in the class "App". User and email must be the same as set in your Github account, the startDate should be less than a year ago.
  4. Execute the main app.
  5. Push the repository to github

HowTo undo

Delete the repository, in our exampel "paper" from your account as described here.
